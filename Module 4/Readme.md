## ğŸ“Š Key Learnings & Insights

- **K-Means:** Efficient centroid-based clustering algorithm for convex data.  
- **DBSCAN:** Detects arbitrary-shaped clusters, robust to noise.  
- **HDBSCAN:** Adaptive density-based algorithm requiring fewer parameters.  
- **PCA:** Linear method for dimensionality reduction and feature compression.  
- **t-SNE & UMAP:** Nonlinear visualizers for high-dimensional datasets.  
- **Feature Engineering:** Use clusters and principal components to improve ML models.
